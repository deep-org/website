Background

The package TREX cleans, processes and visualises highly resolved time series of sap flow data

Importa data (L0)

Raw dendrometer and temperature data can be provided as input. Providing temperature data is optional. However, temperature data increases the quality of error detection and processing.

The raw dendrometer or climate data can be provided in long format. Data of multiple sensors can either be specified in a column named series to separate the sensors.

Dendrometer data always has to be provided in microns. In addition, a column named ts with timestamps in any standard date format (default is date_format = "%Y-%m-%d %H:%M:%S") is required.
